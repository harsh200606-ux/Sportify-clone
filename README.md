# 🎧 Spotify Clone

A basic **Spotify-inspired music player** built using **HTML, CSS, and JavaScript**.  
This project focuses on core audio playback functionality with a clean and simple UI.

> ⚠️ Note: This is a work-in-progress project. Songs are playable, but full features are not implemented yet.

---

## 🚀 Features
- ▶️ Play & Pause songs  
- 🎵 Playable song list  
- ⏱️ Basic progress tracking  
- 🎧 Simple UI  
- ⚡ Lightweight web app

- HTML

- JavaScript
